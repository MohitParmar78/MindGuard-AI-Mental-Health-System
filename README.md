## в—Ҳ The XAI Engine вҖ” Why This Matters

> *"Trust, but verify."* вҖ” Every prediction MindGuard makes can be traced back to the exact words that caused it.

MindGuard uses **SHAP (SHapley Additive exPlanations)** вҖ” a mathematically rigorous framework rooted in cooperative Game Theory вҖ” to answer:

**"Which specific words made the model predict *Anxiety* over *Depression*?"**

```
Input:  "I have a massive presentation tomorrow and my chest is tight."
                в”Ӯ                    в”Ӯ                        в”Ӯ
                в–ј                    в–ј                        в–ј
           [neutral]           [HIGH IMPACT]            [HIGH IMPACT]
           SHAP вүҲ 0.01         "presentation"           "chest is tight"
                               SHAP = +0.43              SHAP = +0.51
                                    в”Ӯ                        в”Ӯ
                                    в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”¬в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
                                               в–ј
                                    Predicted: ANXIETY (87.3%)
                                    Risk Level: MEDIUM вҡ пёҸ
```

This is rendered as an **interactive HTML report** embedded directly in the chat вҖ” red highlights push the prediction *toward* the emotion, blue highlights push *against* it.

## в—Ҳ Quick Start

### Prerequisites

```bash
python >= 3.10
```

### 1 В· Clone & Install

```bash
git clone https://github.com/MohitParmar78/MindGuard-AI-Mental-Health-System.git
cd MindGuard-AI-Mental-Health-System
pip install -r requirements.txt
```

### 2 В· Configure Secrets

Create a `.env` file at the project root:

```env
GROQ_API_KEY=your_groq_api_key_here
```

> Get your free Groq API key at [console.groq.com](https://console.groq.com)

### 3 В· Ensure Model Weights Exist

The XLM-RoBERTa fine-tuned weights must be present at:

```
artifacts/
в””в”Җв”Җ xlmr_weights/
    в””в”Җв”Җ final_mindguard_model/
        в”ңв”Җв”Җ config.json
        в”ңв”Җв”Җ pytorch_model.bin
        в”ңв”Җв”Җ tokenizer_config.json
        в””в”Җв”Җ vocab.json
```

> Train your own using the notebooks in `notebooks/` or download pretrained weights.

### 4 В· Launch

```bash
cd app
streamlit run main.py
```

Navigate to `http://localhost:8501` in your browser.
